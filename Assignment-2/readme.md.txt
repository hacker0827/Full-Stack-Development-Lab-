Assignment 2 - Personal Portfolio Website
##Description

This assignment is a modern personal portfolio website built using HTML,
CSS, and Bootstrap. It showcases personal details, skills, education,
projects, and contact information with a clean and interactive UI design.

The website includes animations, responsive layout, and a professional
developer portfolio structure suitable for real-world use.

##Portfolio Highlights
modern responsive UI using Bootstrap framework
animated sections using AOS (Animate On Scroll) library
hero section with typing animation effect
sections for about, skills, education, projects, and contact
interactive project cards with hover effects
smooth scrolling navigation bar
social media integration in footer

##Technologies Used
HTML5
CSS3

##Files
index.html - main portfolio structure, styling, and scripts

## How To Run 
Open index.html in any browser.
