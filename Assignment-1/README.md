# Assignment 1 - Student Table and HTML Tags Table

## Description

This assignment demonstrates the use of a basic HTML page structure and table
elements. The page includes:

- a student table showing sample student IDs, names, and departments
- an HTML tags table showing common tags, syntax examples, and their purpose

## Features

- semantic HTML structure
- responsive table layout
- hover effect on table rows
- improved spacing, alignment, and color styling
- clean sectioned design using plain HTML and CSS

## Technologies Used

- HTML5
- CSS3

## Files

- `index.html` - page content
- `styles.css` - page styling

## How To Run

Open `index.html` in any browser.
