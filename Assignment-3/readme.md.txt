Fashion Store Website (ModeMuse)
Description

This project is a modern fashion e-commerce website created using HTML, CSS, and JavaScript.
It allows users to browse products, search items, add products to cart and wishlist, and explore different categories.

Features
Responsive design using Bootstrap
Product search and filtering
Add to cart and wishlist functionality
Sorting products by price and rating
Interactive UI with modern design
Technologies Used
HTML
CSS
JavaScript
Bootstrap
Files
index.html – main structure of the website
style.css – styling and layout
script.js – functionality and logic
How To Run

Open index.html in any web browser.
